#### 🔹 Bubble Sort Algorithm
A sorting algorithm that repeatedly compares adjacent elements and swaps them if they are in the wrong order until the array is sorted.

#### 🔹 Linear Search
A searching algorithm that traverses the array sequentially to find a given element.

#### 🔹 Linear Search with Multiple Indices
An extension of linear search that finds and displays all positions where a target element occurs in the array.

#### 🔹 Print Array in Reverse Order
A program that displays the elements of an array from the last index to the first index.
